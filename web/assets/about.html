Go Links

To create a link type go/ and then your prefered link shortcut name. For example, go/finance-dashboard. If the link is already taken then you'll be automatically redirected to the website. If it's not taken then you can paste in the URL that the shortlink will redirect to.

To see a list of all the links go to go/links

To change the location of a shortlink go to go/edit/shortlink (replacing shortlink with your actual value). For example, go/edit/finance-dashboard, which will let you change the URL that the short link redirects to.
